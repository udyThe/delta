selectors like 'h' we use selectors in css to select those on which we want to apply style on

1. element selectors are those selectors which we were using till now like 'h' which selects all headings
2. id selectors - as we assign id to components in html we can select them ...they cannt start with numbers...they may contain but cannnot start wiht numverusing id by - #-id-
3. class selectors - we can define class name in inline style in html with the elemect we wanna group as class ....just assign class in the elect 'h' or whatever
4. decendant selectors - they are like parent child relation.....like div < p then it will only select those paragraph with div in them

sibling combinator

if 'h3 + p' is written then only those paragraph will be selected which are writtedn just after h3 headings
here '+' is the combinator

child combinator

if 'span > button' is written that means it will select buttons only which are directly after the span ...unlike decendent selectors which can skip any number of element
here '>' is the combinator

}

psudo class

a keyword that specifies the special state of the selected element
some types are mentioned
1. hover
2. active
3. checked
4. nth-of-type

to define a psudo class we use .... colon (:) after the specofoed element

1. hover

it is a style which is applied to the the element when hovered by mouse cursor
but in the case in css when some property is already defined using # or id ....then this does not take effect ....its like priority

2. active

it is a style which is applied when a element is clicked on ... not hovered

3. checked

4. nth-of-type

for example: like if we want to color every odd number from a list to yellow....we use nth-of-type

div:nth-of-type(2n){
background-color: blanchedalmond;
}

so the 'n' here works as repeation ...it counts to 2 and then starts again counting from 1
