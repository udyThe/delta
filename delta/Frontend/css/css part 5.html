Flexbox

it is a one dimentional layout method for arranging items in rows or columns. this is used to make the website resposnive and adjust according to the screen available for it.... a flex model is used in this ...in which we give the element flex properties and we emagine axis across the screen ...pretty confusing rn will learn ahead

display flex

the flex function is defaultly works in row direction we can specify the direction by 'flex-direction'...it fits everything inside the parameters

flex direction

it sets how the flex items are placed in the flex container, along which axis and direction...
flex-direction: row; the default main axis is left to right
flex-direction: row-reverse; this makes it to right to left
flex-direction: column; default main axis is top to bottom
flex-direction: column-reverse; this makes it bottom to top


like the ements are arranged accordingly

justify content

in this the content spaceing is handeled ...it also manages the start point too...eg. justify-content: flex-start; (flex-end, cener)

and then theres that which controlls tbhs space ....this space putting works in main axis direction....eg space-between, space-around space-evenly

#main axis is the row in which items are aligned and the 90deg opponent is the cross axis

flex wrap

this is used to wrap the context...suppose the content width is more and the fllex adjusts all element in row/column but u wont want it ...so flex wrap set the desired the size and pushes the next element in next row/column....there are three types,
1. no wrap
2. wrap
3. wrap-reversed which reverses the content across the cross axis which is 90 deg to tha main axis

align items

distributes out items along the cross axis. it have some types
1. justify-content: flex-start
2. justify-content: flex-end
3. justify-content: center
4. align-item: baseline => in this the priority is given to the content to the content written in it rather than the size of div or anything

align content

it distributes the space between and around the content items along the flexbox's cross axis .... there are 4 types
1. align-content: flex-start/ flex-end/ center
2. align-content: space-between/ space around/ evenly
3. align-content: baseline

align self

till now we used to align the container not the individual ones.... but with this we can align the individual ones too along cross axis .... there are 4 types
1. align-self: flex-start
2. align-self: flex-end
3. align-self: center
4. align-self: baseline
it is just line align-items the only difference is the this is applied to an individual and that is applied to the group
the priority of the self one is greater than the group one

felx sizing

it is the assignment of size to the items in the container

flex-basis

it sets the initial main size of the flex item....here the main size mean the height/width according to the rox or column assignment of the flex-direction....if its row then main size would be width ....and vice versa

flex grow

it specifies how much of the flex container's remaining space should be assigned to the flex item's main size .... ex- flex-grow: 1
here the '1' is used for proportions ... like if not defined for any other ...then the one specified with this will take up the whole remaining space in the container...in this min-width: 100px can also be set and max-width: 400px could also be set

flex shrink

it sets the skrink factor of the flex item.... it is also set in the form of '1' ...ex flex-shrink: 1; ....this means if the items are overflowing in size ...then it would shrink....if the shrink factor is given in factor of 1 and a specific element is givenm 2..... then it would be 2 factor smaller then others

flex shorthand

this is used to declare all stuff like grow shrink basis in just one line
...first comes grow then shrink then basis
....ex flex: 2 3 100px; ... in this the grow is 2 and shrink is 3 and basis is 100px
