for setting opacity ...use the term 'opacity' and the value varies from 0 to 1 .... the difference between the using alpha and opacity is that alpha decreases the visibility of only color it is set to ...but the opacity element just chages the visibility of the whole elements included in the portion...like is div is set with opacity then the whole div will be getting changes in the visibility

position property

4 types
1.static
2. relative
3.absolute
4.fixed

static

-static means default value ...no meaning of left right top or bottom offset (shifting from left/rifght to the desired location)
ex
    position: static;

relative

the offset is relative to itself based in the values if top right bottom and left ...like its shifted in respective of itself instead of the webpage
ex
    position: relative
    top: 100px;
    bottom: 100px;

absolute

in this the position is set with respective with the page not with anything , but if the parent class defined then the property is inherited..... if not then it will be positioned respected to the page
ex
    position: absolute;
    top: 200px;
    bottom:200px;

fixed

in this the position is fixed on the screen of pafe and it is fixed even if scrolled...its not blown away with the others when scrolled
ex
    position: fixed;
    top: 200px;
    bottom:200px;
